# ECC Media Platform

ECC Media est un laboratoire journalistique étudiant qui combine journalisme traditionnel et nouveaux médias pour offrir une couverture approfondie de l'actualité universitaire, des innovations scientifiques et des tendances culturelles.

## Structure
- `/admin` - Admin dashboard
- `/backend` - Python Flask API
- `/frontend` - React frontend application

## Installation
1. Set up backend: `cd backend && pip install -r requirements.txt`
2. Set up frontend: `cd frontend && npm install`

## Running
1. Start backend: `cd backend && python app.py`
2. Start frontend: `cd frontend && npm start`
